setup ( )

ESP32/Arduino style entry point. This is ran ONCE by the 'system'.

Setup serial port, format the flash if FORCE_FORMAT is defined. Then either call doActions or playlistAction to exercise the library.

Note
This is a bit unusual as loop is where Arduino-style systems usuall do all their 'work'. But for this test-jig we only do these things once.
