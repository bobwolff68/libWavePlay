Detailed Description

Support for reading using (older deprecated but popular) SPIFFS.

The base class WaveFileBufferReader has 4 pure virtual functions which need to be implemented by these support classes. This allows for the main classes to not be concerned with the implementation specifics of each type of filesystem. The filesystem specifics are /hidden/ under these classes.

Note
SPIFFS is a deprecated library for Arduino/ESP32. Moving to LittleFS is recommended.
