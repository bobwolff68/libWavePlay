\doxysubsection{Detailed Description}
Organizer/\+Manager class for enabling playback of audio files from a list. 

Features include\+:
\begin{DoxyItemize}
\item Scanning for files in an \+\_\+init\+Loc location to gather the full set of files.
\item Features the ability to set one of the audio files as an /\+Intro/. When there is an intro file, playout of any audio file will be preceeded by the playout of the intro.
\item Handles the statefulness of playing audio files including the Intro audio
\item Thread-\/based so that it can handle stateful transitions and playout without any external polling or \textquotesingle{}babysitting\textquotesingle{} of the class.
\item Volume controls via software -\/ not controlling an amplifier
\item Allows for a pin to be utilized for controlling an amplifier as an on/off mechnism. The purpose here is about power savings. If the audio is primarily unused, this allows the amplifier to be turned off using a transitor or MOSFET to do so in hardware.
\item Amplifier control pin can be active high or low via \+\_\+on\+Pin\+High
\item \mbox{\hyperlink{class_audio_playlist_manager_ab1f9d20ddeba890238960a8a3055dcbe}{Play\+Random\+Entry()}} simply picks an entry from the list at random
\item Audio file list can be managed via \mbox{\hyperlink{class_audio_playlist_manager_a734225f48ff565b5eac3157580b25294}{Clear\+File\+List()}} and \mbox{\hyperlink{class_audio_playlist_manager_aa048ddf99c20bcffbbb70e8132b71620}{Add\+Files\+From()}} 
\end{DoxyItemize}
