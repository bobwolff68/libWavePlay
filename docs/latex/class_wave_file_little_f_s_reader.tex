\doxysubsection{Detailed Description}
Support for reading using stdio (for use in native builds) 

The base class \mbox{\hyperlink{class_wave_file_buffer_reader}{Wave\+File\+Buffer\+Reader}} has 4 pure virtual functions which need to be implemented by these support classes. This allows for the main classes to not be concerned with the implementation specifics of each type of filesystem. The filesystem specifics are /hidden/ under these classes.
