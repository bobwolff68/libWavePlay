\doxysubsection{Detailed Description}
Workerbee class for handling a single audio file -\/ parsing and buffer management. 

This is the base class for the other Wave\+File$<$filesystem$>$Reader classes. Features\+:
\begin{DoxyItemize}
\item Thread-\/based WAVE file processing to manage buffer fullness without external controls.
\item Reduce or eliminate popping sound at the speaker by use of the ramp\+Timing which uses the 8-\/bit DAC value of 0x7f as the \textquotesingle{}zero point\textquotesingle{}
\item Buffer filling attempts to be smart about how much it fills at any given call for \mbox{\hyperlink{class_wave_file_buffer_reader_a857a9875fb1dc259519434244e937637}{buffer\+Fill()}} and splits its filling/reading operations into at most 2 operations which might straddle the end of the buffeer (wrap-\/around case)
\item Buffer size allocation is based upon byte\+Rate, number of channels, resolution and rate.
\item Fairly complete WAV header processing ability in order to support as wide a variety as possible of PCM-\/based WAV / RIF files. 
\end{DoxyItemize}
