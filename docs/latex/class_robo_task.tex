\doxysubsection{Detailed Description}
Abstract superclass of tasks (run on a separate thread). Call \mbox{\hyperlink{class_robo_task_a19fb241824ebed8d04e4b43c3ffa1f25}{Start()}} to begin the task and \mbox{\hyperlink{class_robo_task_a4b0430ec1fbcb21b7d57219b7f25d764}{Pause()}} to temporarily pause it. The inheriting class must implement \mbox{\hyperlink{class_robo_task_ac3cffeb81e0bf4ecd03eb2a74dd9022a}{Run()}} which will be called again and again when the task is in \textquotesingle{}Start/\+Running\textquotesingle{} mode.

\end{DoxyNote}
