\doxysubsection{Detailed Description}
helper for supporting try/catch in file-\/based exceptions. 

Allows tracking partially completed file operations and being able to query how many bytes were completed at the time of the exception. This is the typical situation where files are read in \textquotesingle{}blocks\textquotesingle{} or \textquotesingle{}chunks\textquotesingle{} of a fixed size and at the end of the file, there is a \textquotesingle{}remainder\textquotesingle{}.
